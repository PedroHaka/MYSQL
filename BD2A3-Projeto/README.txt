  Este repositório contém os arquivos .sql que compõem meu projeto semestral da disciplina de Bancos de Dados 2.
  Não é perfeito ou complexo, é bem simples na verdade, e as motivações para realização do projeto seriam o aprendizado
e a fixação das habilidades em criar e manipular bancos de dados relacionais utilizando linguagem MySQL.
  Minha maior dificuldade ao implementar este projeto foi conseguir atribuir e referenciar os campos corretos em
views, functions, stored procedures e triggers, e atribuo esta dificuldade à minha baixa familiaridade com a 
sintaxe MySQL, uma bagagem de conhecimentos consideráveis de álgebra relacional foi imprescindível para os resultados
obtidos.
  Por fim, obrigado pela atenção e por conferir o resultado do projeto!
